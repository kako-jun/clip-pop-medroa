# 概要

## クリップボードの変化を見逃さない

何かをコピーしたはずなのに、別の操作でいつの間にか上書きされていた。
コピーできていたか不安で、何度も確認のために貼り付けてしまう。
そういった「コピーの不確かさ」を解消したいという願望から生まれたアプリです。

クリップボードの内容が変わった瞬間に、画面の隅にそっとバナーが浮かび上がる。
「今コピーされた」「今クリアされた」という事実を、作業の流れを断ち切らずに伝えます。

## 何をするアプリか

常駐型のオーバーレイアプリです。起動しておくだけで、クリップボードを監視し続けます。
コピー操作が起きると通知バナーが画面の隅に現れ、しばらくすると自動的に消えます。
他のウィンドウの手前に常に表示されるため、どのアプリを使っていても通知を見逃しません。

## 何をしないアプリか

クリップボードの履歴を保存したり、内容を管理したりする機能はありません。
クリップボードマネージャーではなく、「今この瞬間に何が起きたか」を知らせるシグナルです。

## 名前について

「Clip Pop Medroa」の "Medroa" は造語です。意味より語感を重視して付けられました。
