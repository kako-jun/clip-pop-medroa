# 機能

## トースト通知

クリップボードに変化が生じると、画面の隅に半透明のバナーが浮かび上がります。
「コピーしました」「クリアしました」の2種類のイベントを区別して表示します。
バナーは設定した秒数が経過すると自動的にフェードアウトします。

## テーマ切替

3種類の見た目から選べます。

- **ダーク** — 黒系の半透明背景。暗い作業環境に溶け込みます
- **ライト** — 白系の半透明背景。明るいデスクトップに向いています
- **カスタム** — 自分で用意したPNG/WebP画像を表示します。アニメーション透過WebPにも対応しています

## 多言語対応

アプリはOSの言語設定を自動的に読み取り、対応するロケールファイルを探します。
日本語と英語が同梱されており、見つからない場合は英語に自動フォールバックします。
設定ディレクトリの `locales/` フォルダに独自のYAMLファイルを置けば、任意の言語を追加できます。

## 設定のカスタマイズ

設定パネルから以下を変更できます。変更はリアルタイムでプレビューに反映されます。

| 設定項目 | 内容 |
|---|---|
| テーマ | ダーク / ライト / カスタムの切替 |
| 表示時間 | バナーが消えるまでの秒数（1〜10秒） |
| 表示位置 | 画面の4隅から選択 |
| カスタム画像 | コピー・クリアそれぞれの画像ファイルを指定 |

設定内容はYAMLファイルとして保存され、次回起動時にも引き継がれます。

## 常時最前面表示

オーバーレイウィンドウは常に他のウィンドウより手前に表示されます。
フレームレスかつ透明なウィンドウとして実装されており、デスクトップに自然に溶け込みます。

## ホバー中の一時停止

バナーにマウスカーソルを乗せると、自動消去のタイマーが一時停止します。
カーソルを離すと残り時間ではなく、設定した表示時間からカウントダウンが再開します。
この間に「︙」メニューを開いて設定を変更したり、アプリを終了したりできます。
